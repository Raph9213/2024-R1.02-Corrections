Objectifs

En premier lieu, cela permet de comprendre le concept de séparation des préoccupations. Un fichier HTML ne doit pas contenir d'informations de mise en forme. Le second objectif est d'apprendre comment s'effectue la liaison d'un fichier CSS à un fichier HTML. Et enfin, pour appliquer les styles pour les éléments HTML dans le fichier CSS, il faut utiliser les sélecteurs.

Un exemple de sélecteur à utiliser : 

                        
p {
    font-size: 1.2em;
}

Il faut absolument changer ce genre de chose
